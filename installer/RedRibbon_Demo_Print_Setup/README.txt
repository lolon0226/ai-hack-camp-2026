RedRibbon Demo Print Receiver
=============================

본선 시연용 Print Receiver (설치 경로: C:\RedRibbonDemo)
기존 C:\RedRibbonPrint 와 별도로 동작합니다.

1. install_redribbon_demo.ps1 을 관리자 PowerShell에서 실행
2. SETUP_VIRTUAL_PRINTER.md — PDFCreator 프린터 redribbon 자동저장 폴더 설정
3. C:\RedRibbonDemo\run_redribbon_receiver.ps1 실행
4. print_receiver\check_receiver_ready.ps1 로 점검

서버: http://127.0.0.1:8000
API: /api/print-receiver/upload
